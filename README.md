# TimeCluster
Implementation of the tool used in the paper "TimeCluster: dimension reduction applied to temporal data for visual analytics" by Mohammed Ali, Mark W. Jones, Xianghua Xie and Mark Williams

Here is the paper [Paper](https://link.springer.com/content/pdf/10.1007/s00371-019-01673-y.pdf) and some videos that could give the tool's features: [Video 1](https://www.youtube.com/watch?v=arA3XuXZ7OQ), [Video 2](https://www.youtube.com/watch?v=M64Pk5fz-UM) and [Video 3](https://www.youtube.com/watch?v=ZwLR6fX3Uk4).
